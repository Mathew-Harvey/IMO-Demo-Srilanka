# Sri Lanka National Technology Demonstration & Training Workshop — Participant Portal

A simple, client-side **participant portal** for the **Sri Lanka National Technology
Demonstration & Training Workshop**, held under the **IMO–Norad TEST Biofouling Project**.

- **When:** 23–26 June 2026
- **Where:** Colombo Dockyard / Port of Colombo, Sri Lanka
- **Delivered by:** IMO TEST Biofouling Project Coordination Unit (PCU), MTCC Africa (JKUAT),
  the Marine Environment Protection Authority (MEPA) of Sri Lanka, and international technical
  experts and technology providers.

No login, no install — just open `index.html`.

## What's inside

| Tool | Path | Description |
|------|------|-------------|
| **Participant portal** | `index.html` | Branded landing page linking the workshop tools |
| **Hull Fouling Cost Calculator** | `calculator/index.html` | Estimates fuel, cost and emissions penalty from hull fouling |
| **Demo Hull Inspection Report** | `report/index.html` | Live ROV inspection report from the workshop demonstration |
| **BFMP Generator** | `bfmp/index.html` | Vessel-specific Biofouling Management Plan wizard |

## Hosting

Pure static HTML/CSS/JS. Deploy with **GitHub Pages** (Settings → Pages → deploy from
`main`, root) or any static host. A `.nojekyll` file is included so all assets are served
as-is.

## Reporting invasive marine species in Sri Lanka

Report suspected invasive marine species to the **Marine Environment Protection Authority
(MEPA)** — Tel +94 11 2554373 / +94 11 2554006, info@mepa.gov.lk, [mepa.gov.lk](https://mepa.gov.lk).
